## 🏗️ Featured Project

### 🔥 AWS Three-Tier Architecture (Production-Level)

> A **secure, scalable, and highly available** 3-tier cloud architecture built on AWS following industry best practices — no SSH, no hardcoded credentials, fully isolated network layers.

```
┌──────────────────────────────────────────────────┐
│                   PUBLIC SUBNET                  │
│           Application Load Balancer (ALB)        │
└─────────────────────┬────────────────────────────┘
                      │
┌─────────────────────▼────────────────────────────┐
│                  PRIVATE SUBNET                  │
│           Web Tier  →  App Tier (Node.js)        │
│              EC2 instances via SSM               │
└─────────────────────┬────────────────────────────┘
                      │
┌─────────────────────▼────────────────────────────┐
│               ISOLATED DB SUBNET                 │
│               Aurora (Multi-AZ RDS)              │
└──────────────────────────────────────────────────┘
```

| Feature | Implementation |
|---|---|
| 🔐 Zero SSH Access | AWS SSM Session Manager |
| 🔑 No Hardcoded Credentials | IAM Roles & Instance Profiles |
| 🌐 High Availability | Multi-AZ Deployment |
| 📦 App Deployment | S3 Bucket |
| 🛡️ Network Isolation | VPC with Public/Private Subnets |

**📂 [View Repository →](https://github.com/aadityapalsagwan/three-tier-architecture-aws-ssm)**
