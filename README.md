## 🚀 Featured Project (Highlighted)

### 🔥 AWS Three-Tier Architecture (Production-Level)

> Designed and deployed a **secure, scalable, and highly available 3-tier architecture** on AWS using industry best practices.

**🔧 Tech Used:**
AWS VPC, EC2, ALB, Aurora, IAM, S3, SSM, Nginx, Node.js

**✨ Key Highlights:**

* ✔ Multi-AZ Architecture (High Availability)
* ✔ Public & Private Subnet Isolation (Security Best Practice)
* ✔ No SSH Access → Used **SSM Session Manager**
* ✔ IAM Role-based secure access (No hardcoded credentials)
* ✔ S3 used for application deployment
* ✔ Web, App & DB layers fully separated
